# Parallel Tracking And Mapping (PTAM)

[项目主页](http://www.robots.ox.ac.uk/~gk/PTAM/)

## 主要原理是: 
      从摄影图像上捕捉特征点，
      然后检测出平面，
      在检测出的平面上建立虚拟的3D坐标，
      然后合成摄影图像和CG。
      其中，独特之处在于，立体平面的检测和图像的合成采用并行处理。

## 优点：
      提出并实现了跟踪与建图过程的并行化，
      将前后端分离，使用非线性优化方案，
      既可以实时的定位与建图，
      也可以在虚拟平面上叠加物体。

## 缺点：
      场景小，跟踪容易丢失。
